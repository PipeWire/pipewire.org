Getting PipeWire

Pipewire is available on github. To clone the repository, just 

  $ git clone https://github.com/PipeWire/pipewire.git

Setup

Building pipewire is fairly straightforward. 

  $ ./autogen.sh --prefix=$PREFIX
  $ make
  $ make install
The $PREFIX is commonly set to /usr or /usr/local

Get Involved

PipeWire is Free Software and is developed in the open. To get in touch, use the pipewire mailing list or #pipewire IRC channel on Freenode. Code can be found on Github (along with a wiki) and issues are tracked on Github Issues.
